## ğŸš€ Getting Started

### Prerequisites
- Microsoft Power BI Desktop
- Power BI Service account (for publishing)
- Dataset: Future Jobs Market Data

### How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/Thiru072000/PowerBI-Dashboard-Project.git
   ```

2. **Open the dashboard**
   - Navigate to the `dashboards/` folder
   - Open `Future Tech Job_Overview.pbix` with Power BI Desktop

3. **Review the data**
   - Check the `data/` folder for the dataset
   - Dataset: `future_jobs_dataset.csv`

4. **Publish to Power BI Service**
   - Follow the detailed instructions in `PUBLISHING_GUIDE.md`
