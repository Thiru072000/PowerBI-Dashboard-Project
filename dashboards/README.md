# Dashboards Folder

This folder contains Power BI dashboard files (.pbix).

## Files

- `Future Tech Job_Overview.pbix` - Power BI dashboard file containing visualizations and analysis of future job market trends.

## Dashboard Description

The Future Tech Job Overview dashboard provides insights into:
- Job market trends across different industries (AI, Blockchain, Quantum Computing, Green Tech)
- Salary analysis by location and job title
- Remote work availability statistics
- Company size distribution
- Skills demand analysis
- Geographic distribution of job opportunities

## How to Use

1. Download the .pbix file
2. Open it with Power BI Desktop
3. Refresh the data connection to see updated insights
4. Publish to Power BI Service following the PUBLISHING_GUIDE.md

## Requirements

- Power BI Desktop (Latest version)
- Access to the data source (future_jobs_dataset.csv in the data folder)
- Power BI Pro or Premium license for publishing to Power BI Service
