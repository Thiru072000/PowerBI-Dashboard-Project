# Data Folder

This folder contains datasets used in the Power BI dashboard.

## Files

- `future_jobs_dataset.csv` - Dataset containing future job market data with information about job titles, industries, locations, salaries, skills required, and remote options.

## Dataset Description

The future jobs dataset includes the following columns:
- Job ID
- Job Title
- Industry
- Location
- Salary (USD)
- Skills Required
- Remote Option
- Company Size
- Posting Date

## Usage

This dataset is used as the primary data source for the Power BI dashboard visualization.
